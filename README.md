
# Requirements

- Any nerd font for the terminal
- [`ripgrep`](https://github.com/BurntSushi/ripgrep?tab=readme-ov-file) 
- `GCC`

# Keybindings

- All keybinds are in keymaps.lua
- Defaults may not be explicitly defined

# Backend

- When adding languages, it is necessary to add to `lua/plugins/lsp/mason.lua`
- If adding another plugins folder, it must also be added to lazy-nvim.lua
